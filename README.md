# Simple-E-commerce

![thumb](https://github.com/JorgeCJ/Simple-E-commerce/assets/127647774/6303a42e-559c-4a13-801e-95465f252251)

Small and simple project, but complete with front-end and back-end. The idea is to simulate a simple subscription to a given service. The project was made on Youtuber [Fernanda Kipper](https://www.youtube.com/watch?v=lq7b8uZULs4)'s live, but with some visual modifications that I made.

## Technologies used:
- [Javascript](https://developer.mozilla.org/en-US/docs/Web/JavaScript)
- [Typescript](https://www.typescriptlang.org/docs/)
- [React](https://react.dev/learn)
- [Vite](https://vitejs.dev/)
- [NodeJs](https://nodejs.org/en/docs)
